مقدمة

نرحب بك في تطبيق وطن سداد - شحن فوري و باقات ذو اسم الحزمة (watan.sadad). نحن ندرك أهمية خصوصيتك ونلتزم بحماية بياناتك الشخصية. توضح سياسة الخصوصية هذه كيفية جمعنا واستخدامنا وحماية معلوماتك الشخصية عند استخدامك لتطبيقنا.

باستخدامك للتطبيق، فإنك توافق على جمع واستخدام المعلومات وفقًا لهذه السياسة. نحن لا نشارك معلوماتك مع أي أطراف ثالثة إلا كما هو موضح في هذه السياسة.

البيانات التي نجمعها

نقوم بجمع أنواع مختلفة من المعلومات لتقديم خدماتنا وتحسين تجربتك:

البيانات التي تقدمها مباشرة

  • بيانات التسجيل: الاسم الكامل، رقم الهاتف، العنوان (كتابة يدوية)، اسم المستخدم، وكلمة المرور.
  • بيانات الخدمات: المعلومات التي تدخلها عند طلب خدمة معينة.

البيانات التي نجمعها تلقائياً

  • بيانات الاستخدام: معلومات عن كيفية تفاعلك مع التطبيق والخدمات التي تستخدمها.
  • بيانات الجهاز: نوع الجهاز ونظام التشغيل وإصدار التطبيق.

البيانات التي نطلب صلاحيات الوصول إليها

  • الكاميرا: نطلب صلاحية الوصول للكاميرا فقط عند الحاجة لالتقاط صورة أو قراءة الباركود مباشرة.
  • جهات الاتصال: نطلب صلاحية الوصول لجهات الاتصال فقط لاختيار رقم هاتف (لا نرسل الأسماء إلى الخادم).
  • التخزين/الصور: نستخدم تقنية Native لاختيار الصور بدون طلب صلاحية الوصول للصور.

ملاحظة هامة

نستخدم تقنية Native لاختيار الصور ورقم الهاتف، مما يعني:

  • لا نطلب صلاحية الوصول إلى الصور عند اختيار صورة من الجهاز
  • لا نطلب صلاحية الوصول الكامل إلى جهات الاتصال، فقط لاختيار الرقم (بدون الأسماء)
  • لا نرسل أسماء جهات الاتصال إلى الخادم، فقط رقم الهاتف المحدد

كيف نستخدم بياناتك

  1. توفير الخدمات المطلوبة وتنفيذها.
  2. إنشاء حسابك وإدارته.
  3. تحسين أداء التطبيق وتجربة المستخدم.
  4. التواصل معك بشأن خدماتك أو تحديثات التطبيق.
  5. ضمان أمن التطبيق وحماية بيانات المستخدمين.

معالجة البيانات المؤقتة

  • يتم تخزين البيانات اللازمة لإتمام الخدمة فقط أثناء فترة المعالجة.
  • بعد إتمام الخدمة، نحتفظ بالبيانات المطلوبة للسجلات المالية فقط وفقاً للفترة القانونية المطلوبة.
  • لا نحتفظ بالصور المرفوعة بعد إتمام الخدمة إلا إذا كانت ضرورية للإثبات أو للأغراض القانونية.

مشاركة البيانات مع أطراف ثالثة

نحن لا نبيع أو نؤجر بياناتك الشخصية للآخرين. قد نشارك بعض البيانات المحدودة مع جهات خارجية موثوقة (أطراف ثالثة) لتسهيل تشغيل التطبيق وتقديم الخدمات، وهم ملتزمون تماماً بحماية خصوصيتك وعدم استخدام البيانات لأي أغراض أخرى. تشمل هذه الأطراف:

  • خدمات Firebase و Google Analytics: لتحسين أداء التطبيق، وتشخيص الأعطال، وإرسال الإشعارات الفورية للمستخدمين.
  • مزودي خدمات الاتصالات وبوابات الدفع الإلكتروني: لمعالجة عمليات السداد وشحن الرصيد والخدمات المالية المتاحة داخل التطبيق.
